# Azure VM Deployment Lab

## Objective
Deploy a virtual machine and establish secure access.

## Steps
- Created a Resource Group
- Deployed a Virtual Machine
- Configured login credentials
- Connected via RDP/SSH

## Validation
- Successfully connected to VM
- Verified VM is running

## Key Learnings
- Resource groups help organize resources
- VM deployment requires proper region and sizing

## Improvements
- Add NSG rules for better security
